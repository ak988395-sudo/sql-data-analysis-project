# 📊 Data Analysis using SQL

## 🔍 Overview
This project focuses on analyzing large datasets including sales, flights, and product data using SQL. The goal was to extract meaningful insights and support business decision-making through efficient querying and data transformation.

The project demonstrates strong SQL fundamentals along with advanced querying techniques used in real-world analytics.

---

## ⚙️ Key Features

### 🔹 Data Extraction & Cleaning
- Extracted data from multiple tables
- Cleaned and structured raw datasets for analysis

### 🔹 Advanced SQL Queries
- Used JOINS, Subqueries, and Common Table Expressions (CTEs)
- Applied Window Functions (RANK, DENSE_RANK, ROW_NUMBER)

### 🔹 Data Analysis
- Identified top-performing products and stores
- Analyzed sales trends and revenue drivers

---

## 🛠️ Tools Used
- SQL  
- Data Aggregation  
- Data Transformation  

---

## 📈 Outcome
- Generated actionable insights for business optimization
- Improved understanding of operational and sales performance
